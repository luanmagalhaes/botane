System Flow

A single user message triggers the agent. Claude Sonnet 4.6 decides which tools to call; each tool runs a specific agent and streams its status back through SSE.

Current Agents

These seven are implemented, tested, and running against real Gmail data.

Active agent
Planned agent
📧
Inbox Reader
read_inbox
Connects to Gmail via IMAP and returns the list of B2B emails with sender, subject, partner, and type.
imapflow + mailparser → real Gmail inbox
🔍
Email Classifier
get_email
Fetches a specific email in full including attachments. Identifies the partner and the request type.
Rule-based partner/type detection on subject + body
📄
PO Parser
parse_purchase_order
Extracts structured order data from a purchase order email — PO number, items, quantities, prices, totals.
claude-haiku-4-5 • JSON extraction
🧩
Custom Extractor
extract_structured_order_data
Deep extraction for PDFs and Excel with strict ean/sku, price normalization, and currency validation.
claude-haiku-4-5 • PDF as base64 document
📦
Stock Checker
check_stock
Queries Shopify inventory per SKU. Flags items that are unavailable or do not exist in the catalog.
Shopify Admin API • GraphQL productVariants
🛠️
Order Builder
create_shopify_draft
Creates a real draft order in Shopify, only with in-stock items. Returns the draft ID and invoice URL for human review inside Shopify.
Shopify Admin API • REST draft_orders
✅
Order Verifier
verify_order_total
Confirms the draft total matches the original PO. Flags discrepancies caused by out-of-stock items.
Pure calculation • local
